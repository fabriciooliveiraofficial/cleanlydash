# AirGoverness - Workflow de Desenvolvimento

## Servidor de Desenvolvimento
// turbo-all
```powershell
# Iniciar servidor (use SEMPRE este comando)
npm run dev
```
Acesse: **http://localhost:3000**

> **Hot Reload:** O Vite atualiza automaticamente ao salvar arquivos.
> **Não use Live Server:** O projeto requer transpilação de TypeScript/JSX.

## Build de Produção
```powershell
npm run build; npm run preview
```
Acesse: **http://localhost:4173**
